Anleitung

Vor dem ersten Spielen

Lesen Sie diese Anleitung

Vor jedem Spiel

Legen Sie die Karten gemischt auf den entsprechenden Stapel.
Außerdem sucht sich jeder Spieler eine Spielfigur aus und stellt diese auf das Startfeld.

Spielablauf

Zugablauf

Zum Beginn jedes Zuges dreht jeder Spieler die Drehscheibe und geht die Anzahl an Feldern vor, die auf der Drehscheibe stehen.
Falls ein Spieler auf einem Aktionsfeld landet, zieht er eine Aktionskarte und führt die Aktion aus.
Nachdem jeder Spieler die obigen Aktionen ausgeführt hat, wird eine Karte vom Minispielkartenstapel gezogen und das gezogene Minispiel wird gespielt.
Der Gewinner des Spiels darf in der nächsten Runde doppelt drehen.

Ende des Spieles

Sobald ein Spieler das Ziel erreicht hat, ist das Spiel vorbei.

Viel Spaß beim Spielen!
